About

This is the web user interface of Neubot v0.4.1. Neubot is a lightweight open-source program that runs in background and periodically performs transmission tests to probe your Internet connection using various application level protocols. Read more

Neubot is a research project on network neutrality of the
NEXA Center for Internet & Society at Politecnico di Torino.
